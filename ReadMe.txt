Disini membuat website Uschool yang dapat diakses Guru,Admin, dan Student.
Perlu login saat 
1.Murid ingin melihat nilai
2.Guru ingin mengupdate nilai
3.Admin ingin add user baru atau update data atau delete.

Login membutuhkan input berupa user_id dan password.
user_id dan password dapat dilihat pada database uschool.
berdasarkan role_id , login akan mendirect ke webpage student/teacher/admin sesuai role_id yang didapatkan dari user_id inputan.

Saat ingin pindah role, anda perlu logout terlebih dahulu.
